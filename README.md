# OldTimes

Extensão do Google Chrome para cronometrar e temporizar o tempo voltada para pessoas de meia idade com pouca visão, presença de daltonismo, baixa movimentação e desejam evitar a dependência de terceiros para tarefas simples.

Este projeto fora desenvolvido nas dependências do SENAI Mariano Ferraz sob tutoria do docente [Tiago Neves](https://github.com/tiagosfneves)

## Visão Geral

Este projeto consiste na criação e manutenção de uma extensão acessível para navegadores (Google Chrome), desenvolvida para simplificar o uso das ferramentas cronômetro e temporizador. 

O público-alvo são idosos que enfrentam déficits de visão (daltonismo, baixa visibilidade) e baixa compreensão tecnológica. 

O foco é permitir que o usuário utilize o cronômetro e temporizador simples, intuitivo e acessível para registrar o tempo sem erros, com confiança e evitando depender de terceiros.
